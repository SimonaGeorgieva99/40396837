JavaScript arrays are used to store multiple values in a single variable. Each element can be accessed by referring to an index number.

→

var array_name = [item1, item2, ...];

var arr = new Arrays("item1", "item2", "item3");
var item = arr[0];

The toString() method returns an array as a comma separated string. Another useful thing when working with arrays is popping items out and pushing items into.

→

<body>
<p id="test" > </p>
<script>
var arr = ["item1", "item2", "item3"];
document.getElementById("test").innerHTML = arr.toString();
arr.pop(); (remove item)
document.getElementById("test").innerHTML = arr;
arr.push("item4"); (add items)
document.getElementById("test").innerHTML = arr;
</script>
</body>
