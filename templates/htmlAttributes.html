Attributes provide additional information about the lements and are specified in the start tag. They are used to modify the element by adding some additional features. It provides functionality and some elements cannot function without it. For example the href and the src are always used when we need a link or an image, respectively.

→

< a href="URL to a website" > Website to visite < a >

< img src="path to image" width="size" height="size" >

Another example of an attribute can be the style attribute which is used to style a particular element. The example on the right shows how it is used.
And one last attribute added to this basic knowledge of attributes is called lang. It is used for declaring the language of the webpage and also help the seach engines and browsers.

→

< p style="color:pink;" > This paragraph has an attribute color to make its content pink. < p >

< html lang="en" > - The webpage will be in English.
