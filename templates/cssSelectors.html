CSS has two other type of Selectors: class and id. The class selector, selects HTML elements with a specific class attribute. To select it, it a dot is added before the class name. It can be given to one, two or more elements and can be styled at one place. Here is an example:

→

< style >
.classOfElement {
text-align:center;
color:pink;
}
< /style >

<p class="classOfElement"> This element is centered and in pink. <p>
<p class="classOfElement"> This is another element which is centered and in pink. <p>

Another type of selector is id. It is used for elements that are unique within the page. The id is specified using '#' in front of the id name.

→

< style >
#idOfElement {
text-align:center;
color:pink;
}
< /style >

<p id="idOfElement"> This element is centered and in pink. <p>
<p> This element has no style. <p>
