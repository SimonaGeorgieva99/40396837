Tags in HTML can be used to define headers, content text, images, links, tables, forms and many more things. But we will not be able to explore all of them while studying the basics of web development. The top example on the right is showing how a tag is created using HTML. It consist of a start tag surrounded by < >, content which is the text shown on the screen and an end tag.

→

< tagname > Content of the tag < tagname >

How an HTML page is structured?
We start with <!DOCTYPE html>. Then come the head which contains the title, CSS, JS, etc. It is followed by the body which is the part containing all of the content of the page and finished with a footer.

→

<!DOCTYPE html>
<head> </head>
<body> </body>
<footer> </footer>
</html>
