Variables are containers for storing data values. It is done using the keyword var. Other keywords that are used are let and const.

→

<script>
var x=5;
var y=2;
var z = x + y;
</script>

Variables can hold numbers and text values called strings written with quotes. JavaScript variables are created as following:

→

<body>
<p id="test"> </p >
</body >
<script>
var drink='Milk';
document.getElementById('test').innerHTML = drink;
</script>
