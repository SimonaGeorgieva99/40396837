CSS is an abbreviation of Cascading Style Sheets and it is a style sheet language. It is used for the visualisation of a document in a markup language such as HTML. It enables the saparation of elements and their visual representation. It can add colors, fonts, style text and images in many ways and many more. It improves accessibility and flexibility. Web pages can be easily transformed into something else later by changin the properties in the styles sheet.

Formatting helps in displaying different pages with different format and styles. It has a simple syntax and uses English keywords to specify the names of the properties.
