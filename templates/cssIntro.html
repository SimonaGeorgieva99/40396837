CSS statement consists of a selector and declaration block. The selector is the HTML element you want to style. The declaration block contains property-value pairs separated by ';'

→

h1 {
color:pink;
background-color:blue;
}

The previous example used the CSS element Selector. This can be changed with any other selector which we need to add style to.
The p element can be changed with any other HTML element.

→

p {
// Add code here
}
