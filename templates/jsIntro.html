What can JavaScript do with HTML content?
The answer is multiple things. It can substitute any content and add functionality to it. Here is an example which will switch the content of the paragraph to another when a button is clicked:

→

<p id="test"> Content before the change. </p>

<button type="button" onclick='document.getElementById("test").innerHTML = "Content after the button is clicked"'>Click here!</button>

Another really interesting and useful thing that it can do is change the style of the elements. It is done the following way:

→

<p id="test"> Content before the change. </p>

<button type="button" onclick='document.getElementById("test").style.fontSize='30px' = "Content after the button is clicked and the style is changed"'>Click here!</button>
