Conditional statements: if statement to execute if a condition is true. Else statement specifies a block to be executed if the if statement is false.

→

if (condition) {
// block of code if the condition is true
} else {
// block of code if the condition is false
}

Multiple conditions are done as following:

→

if (condition1) {
// block of code if the condition1 is true
} if else (condition2) {
// block of code if condition1 is false and condition2 is true
} else {
// block of code if both of the statements are false
}
