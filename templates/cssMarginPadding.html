The CSS margin is used to create space around element and to differentiate one from another.It can be set on all of the sides - margin-top, margin-right, margin-bottom, margin-left. They can have a value of auto, length, % and inherit.

It can be all grouped and all 4 properties can be written together: margin: 10px 10px 10px 10px;. It can also have three values (top, right&left, bottom), two values (top&bottom, right&left) and one (all in one).

→

< style >
div {
margin-top:10px;
margin-right:10px;
margin-bottom:10px;
margin-left:10px;
}
< /style >

<body>
<div > This element has a all sides at margin 10px. </div>
</body>

It can be set to auto to horizontally center the element within its container. The space will be equally split between the left and right margins.

If it is set to inehrit, it means that it is inherited from another element and they both end up having the same margin.

→

< style >
div {
margin:auto;
}
< /style >

<body >
<div> The element is horizontally in the center within the div box and has equal values for left and right.</div>
</body>
