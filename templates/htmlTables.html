HTML tables are used for arranging data into rows and columns. Each table has a row, header and data. To make the table more good looking, some styele using CSS should be added. The example on the right will show what tags are used to form a simple table.

→

<table> (creates the table)
<tr> (row of a table)
<th> First name </th> (table header)
<th> Last name </th>
</tr>
<tr>
<td> Simona </td> (data in the table)
<td> Georgieva </td>
</tr>

Sometimes more than one record of data is needed to be put in the table. The 'colspan' and 'rowspan' attributes can be added to the table header by giving it a value which is the number of rows and columns we need to add more. Here is simple example:

→

<table>
<tr>
<th colspan="2"> Phone number </th>
</tr>
<tr>
<td> 07759582458 </td>
<td> 07747365912 </td>
</tr>
