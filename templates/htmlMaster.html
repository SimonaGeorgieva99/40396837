HTML is an abreviation of Hypertext Markup Language. It is used for documents that are used to be presented on a web browser. In addition to it, it can be added a Cascading Style Sheets (CSS) and a scripting languages such as JavaScript. HTML documents are sent to the web browser via a web server or from local storage and then they are rendered into the pages. It describes the structure of a page semantically. By using it and some other tools, you can create your own website!

HTML elements are the ones making the web page full. It can contain different structures, images and other objects. HTML provides a structured document for all items. All of the elements are surrounded by tags which we will expore during the lessons. This tags are then styled using CSS. HTML can embed programs written in a scripting languages which add functionality to it.
