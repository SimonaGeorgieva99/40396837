CSS text can have added color, background-color as we covered before. It can be also aligned in the center or any other direction. It can be tranformed and shadowed. Here is an example of text shadowing:

→

<style>
h1 {
text-shadow: 2px 2px;
}
</style>

<body>
<h1> Text with effect. </h1>
</body>

The CSS font properties define the font family, boldness, size and style. It can be from the Serif or Sans-serif family. font-size is simply used by adding the value which we need.

→

<style>
h1 {
font-family: Impact, Charcoal, sans-serif;
}
</style>

<body>
<h1> Text with different font. </h1>
</body>
