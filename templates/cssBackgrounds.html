CSS background property is used to define the background effects. Some of them are background-color, background-image, etc. Background-color specifies the color of the whole page or any box. There are other additional features such as opacity which can be set as well.

→

< style >
body {
background-color:lightcyan;
}
< /style >

<body >
<p> This element is in the body and the page is in lightcyan. <p>
</body>

Background-image specifies an image that is used as a background of an element. When adding a picture, you should be sure that it does not disturb the image and make it unreadable.

→

< style >
body {
background-image:url("pathToImage");
}
< /style >

<body >
<p> This element is in the body and there is a picture behind it.<p>
</body>
