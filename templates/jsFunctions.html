The function keyword is used to define a function is JavaScript. You can use a function declaration or expression.

→

function name(parameters) {
// code to be executed
}

Functions can have parameter value(arguments). Their data type does not need to be defined and does not perform type checking. If the parameters are missing while calling the function, they are set to undefined by default. Here is an example when they are set by default:

→

<body>
<p id="test">
<script>
function myFunc(x, y = 3) {
return x + y;
}
document.getElementById("test").innerHTML = myFunction(4);
</script>
</body>
