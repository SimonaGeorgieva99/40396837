Loops can execute a block of code a number of times. Useful when the same code needs to be run over and over again. Different kind of loops are for (loops through a block of code a number of times), do/while (loops through a code while a specified condition is true) and while.

→

for (statement1(star point); statement2(end point); statement3(iteration)) {
// block of code to be executed
}

do {
// block of code to be executed
}
while (condition);

Example how it is used with HTML:

→

<body>
<p id="test">
<script>
for (i=0; i<5; i++) {
text += "The number is " + i;
}
document.getElementById("test").innerHTML = text;
</script>
</body>
