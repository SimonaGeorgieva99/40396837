HTML forms are used to get user input which is then sent to server for processing. It is created using the <form> tag and can contain different elements in it such as input, text area, labels, buttons, checkboxes, etc. The example on the right is one of the simplest forms which is used in the 'Contacts' page as well.

→

Checkboxes might also be another really useful thing when creating a form and the following example will give a piece of code showing how to create a checkbox.

→

<form action="#" (where to go after clicking on submit)>
<input type="checkbox" name="drink1" value="Coffee">
<label for="drink1"> I love coffee </label>
<input type="checkbox" name="drink2" value="Milk">
<label for="drink2">I love milk </label>
<input type="submit" value="Submit">
</form>
