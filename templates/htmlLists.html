First type of lists is unordered. The the list item will be printed with bullet points (black dots) by default.

The other type is ordered which marks the elements with numbers by default.

The examples will show how you can use both of them.

→

Unordered list
<ul>
<li> First element </li>
<li> Second element </li>
< /ul >

Ordered list
<ol>
<li> First element </li>
<li> Second element </li>
< /ol >

HTML also supports description lists. These are lists of terms with description of each term. It has three tags: one to define the type of the list, the other defines the term and the last one describes it.

→

Description list
<dl>
<dt> Coffee </dt>
<dd> black hot drink </dd>
<dt> Milk </dt>
<dd> white cold drink </dd>
< /dl >
